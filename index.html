📖 Nápověda — jazyk Karel

🚀 Jak začít za 30 sekund

Program píšeš do editoru vlevo. Každý příkaz napiš na svůj řádek (velká písmena, diakritika). Pak stiskni ▶ Spustit nebo ⏭ Krok pro pomalé krokování.

💡 Tip: město vpravo je upravitelné — v panelu nahoře přepni nástroj na Značka / Stěna / Karel a klikni do mřížky. Klik na Karla ho otočí. Shift+klik značku odebere.

📐 Struktura programu

Program je posloupnost příkazů. Píše se shora dolů a vykonává se ve stejném pořadí. Odsazení (mezery na začátku řádku) nejsou povinné, ale dělají kód čitelnější.

// Jednoduchý program — 3 kroky vpřed a polož značku
KROK
KROK
KROK
POLOŽ

🔧 Základní příkazy

KROK posuň se o 1 políčko vpřed (chyba, když je tam stěna) VLEVO-VBOK otoč se o 90° doleva — zkráceně VLEVO VPRAVO-VBOK otoč se o 90° doprava — zkráceně VPRAVO (vnitřně = 3× vlevo) POLOŽ polož značku na aktuální políčko (taky POLOZ) SEBER seber značku z aktuálního políčka (chyba, když tam není)
✏️ Zkrácené tvary: VLEVO = VLEVO-VBOK, VPRAVO = VPRAVO-VBOK, VLEVO-BOK / VPRAVO-BOK taky fungují. Můžeš kombinovat — obojí zápis je v pořádku.

🔁 Opakování — OPAKUJ

OPAKUJ 4-KRÁT
  KROK
  VLEVO-VBOK
*OPAKUJ

Blok mezi OPAKUJ n-KRÁT a *OPAKUJ se provede n-krát. Místo 4-KRÁT můžeš napsat 4 KRÁT nebo jen 4.

❓ Podmínka — KDYŽ

KDYŽ JE-STĚNA
  VLEVO-VBOK
JINAK
  KROK
*KDYŽ

Větev JINAK je volitelná.

♾️ Smyčka — DOKUD

// dojdi až ke stěně
DOKUD NENÍ-STĚNA
  KROK
*DOKUD

🧠 Vlastní procedury — NAUČ (podprogramy)

Tohle je srdce Karla: pojmenovaný blok kódu, který se dá volat jak kterýkoli jiný příkaz. Tak se v klasickém Karlovi definovaly třeba otočky, tanečky, nebo rozpoznávání vzorů. Definice musí být před svým voláním.

// klasický příklad — naučit Karla "VPRAVO" pomocí 3× vlevo
NAUČ VPRAVO
  VLEVO-VBOK
  VLEVO-VBOK
  VLEVO-VBOK
*NAUČ

// definice může volat další vlastní procedury
NAUČ ČTVEREC
  OPAKUJ 4-KRÁT
    KROK
    VLEVO-VBOK
  *OPAKUJ
*NAUČ

// a teď je používáš jako vestavěné příkazy:
ČTVEREC
VPRAVO
ČTVEREC

Tip: VPRAVO-VBOK je už vestavěný, ale klidně si podobné zkratky vyrobíš pro cokoli — např. NAUČ TANEC …, NAUČ POLOŽ-A-KROK ….

🔍 Podmínky (používají se po KDYŽ / DOKUD)

JE-STĚNA před Karlem je stěna nebo okraj města NENÍ-STĚNA před Karlem je volno JE-ZNAČKA na aktuálním políčku je alespoň jedna značka NENÍ-ZNAČKA na aktuálním políčku není značka JE-SEVER Karel kouká na sever (nahoru) JE-JIH Karel kouká na jih (dolů) JE-VÝCHOD Karel kouká na východ (doprava) JE-ZÁPAD Karel kouká na západ (doleva)

⚙️ Ovládání

  • ▶ Spustit — spustí celý program (rychlost nastavíš posuvníkem)
  • ⏭ Krok — provede jeden další příkaz (dá se mačkat opakovaně)
  • ■ Zastavit — přeruší běžící program
  • ⟲ Reset — vrátí město i Karla do stavu před spuštěním
  • F1 / Esc — otevře / zavře tuto nápovědu
💡 Začínáš? V pravém horním rohu editoru máš rozbalovačku příklad — vyber některý a uvidíš hotové programy s předpřipraveným městem.
